Project Introduction:

This project focuses on visualizing and analyzing hotel booking data using Tableau tools for the Information Technology division at Telkom Indonesia. The goal is to identify patterns and trends in hotel bookings, cancellations, and market segments to improve business decision-making.

Assignment 1: Hotel Booking and Cancellations Visualizations

In task 1, the data visualization process will explain the distribution of hotel reservations based on the hotel country. Then, the process will explain the distribution of hotel order cancellations based on consumer type and market segments. This process will also explain the distribution of hotel order cancellations based on distribution channels.

Assignment 2: Data Exploration and ULDL Visualization

Exploring Order Data by Region:

For this assignment, we explored order data based on regional performance. This visualization helps in understanding regional order trends and distribution.

Assignment 3: Data Visualization with Image

In this task, data visualization is done using images, with two floating sheets: one for total data and two sheets displaying status categories. These visualizations were created in Tableau for clarity and simplicity.

Assignment 4: App Status Visualization

This task involves visualizing the status of the app in real-time. The visualization includes graphical representations of app statuses over time with breakdowns by hour, day, month, and year for easy understanding.

Conclusion:

This project demonstrates the effectiveness of Tableau in visualizing and analyzing complex data, where each assignment successfully utilized its features: Assignment 1 effectively mapped hotel bookings and cancellations, Assignment 2 provided deep insights into regional trends through data exploration, Assignment 3 enhanced visual clarity using dynamic floating sheets, and Assignment 4 offered a user-friendly view of app statuses across various timeframes, ultimately achieving valuable, actionable insights that aid in optimizing business decisions and improving customer experience.
