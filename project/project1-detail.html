Project Introduction:

This project investigates the use of time-based feature expansion methods to enhance the accuracy of classification models in predicting disease spread using SVM and DNN algorithms. We incorporate time-related features from previous periods to refine the predictions. The goal is to create a model capable of forecasting disease spread years in advance for effective public health interventions.

Implementation Process:

Preprocessing:

The initial step is to convert the raw data into a structured format. We check each feature for stationarity and apply differencing to non-stationary features. Once we ensure the datas stationarity, we categorize it into Dengue and COVID-19 datasets and perform normalization to standardize the input data.

Time-Based Feature Expansion:

We enrich the dataset by incorporating time-related features from previous periods, enabling the model to capture temporal disease spread patterns. This approach improves the predictive accuracy, helping to foresee future disease trajectories more effectively.

Data Distribution:

The data is split into training and testing sets using stratified k-fold cross-validation with k = 10, ensuring robust model evaluation and reliable prediction outcomes.

Results and Findings:

The Time-Based Feature Expansion method proved to be highly effective for disease prediction, with SVM showing superior performance compared to DNN.

  • Model 3a (SVM): Achieved 90% accuracy using 7 features from 3 different kernel types.
  • Model 3b (SVM): Achieved 83% accuracy with 4 features (rainfall, temperature, and humidity).
  • COVID-19 Dataset: Model 7j using SVM achieved 93% accuracy with 112 features.
  • DNN model achieved a peak accuracy of 93% using data from the previous 7 months.

The combination of diverse SVM kernels and the complexity of the DNN models played a significant role in achieving these impressive results. This approach lays a strong foundation for public health planning, especially for timely interventions to combat disease outbreaks.
